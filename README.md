# MAC-OS

## Description

This project is a simple example of a REST API. The goal is to learn docker with it.

## Getting Started
### Prerequisites
- Docker 26.1.1
- PostGreSQL
- IntelliJ IDEA 2024.1.1
- Maven 3.9.7
- Java 17

### Configuration

## Deployment

### On dev environment

1. Clone the repository
2. run ```docker-compose up --build```
